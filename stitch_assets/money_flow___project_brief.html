# Project Brief: Money Flow - Modern Financial Mastery ## 1. Project Overview Money Flow is a modern, responsive fintech application designed to help users master their finances. It provides a clean, intuitive interface for tracking income, expenses, liabilities (debts/paylaters), and account balances across both mobile and desktop platforms. ## 2. Target Audience - **Tech-savvy individuals** looking for a premium financial tracking experience. - **Mobile-first users** who need quick transaction entry on the go. - **Power users** who utilize the desktop dashboard for deep financial analysis. ## 3. Core Features - **Unified Dashboard**: Real-time overview of total balance, income vs. expenses, and AI-driven financial insights. - **Transaction Management**: Streamlined history with categorisation, date-based grouping, and search/filter capabilities. - **Category Management**: Customizable expense and income classifications with modern iconography. - **Account Tracking**: Card-based management of various funding sources (Cash, Bank, E-Wallets). - **Debt & Paylater Tracker**: Progress-oriented view for monitoring liabilities and payment statuses. - **Theme Support**: Seamless switching between premium Dark Mode and Light Mode. ## 4. Visual Identity (Lumina Finance Design System) - **Aesthetic**: Modern Fintech / Glassmorphism. - **Primary Color**: Electric Teal (#00f5ff) - Used for primary actions, financial gains, and high-priority data. - **Secondary Color**: Vivid Purple - Used for navigation highlights and AI features. - **Surface**: Deep Navy (#0b1326) for Dark Mode, providing a premium, low-strain backdrop. - **Typography**: Inter (Sans-serif) for maximum legibility across all screen sizes. - **Shape**: Rounded corners (8px) for a soft, modern feel. ## 5. Design Principles - **Clarity First**: High contrast for financial figures to ensure readability at a glance. - **Action-Oriented**: Floating Action Buttons (FAB) for the most frequent tasks (e.g., adding transactions). - **Responsive Symmetry**: Parity between mobile bottom navigation and desktop sidebar navigation. - **Minimalism**: Removal of non-functional elements (e.g., notifications) to keep the user focused on their flow. ## 6. Technical Scope - **Device Support**: Mobile (Portrait) and Desktop (Responsive Web). - **Components**: Shared library including TopAppBar, SideNavBar (Desktop), BottomNavBar (Mobile), and standardized Data Cards. - **Visuals**: CSS-based glassmorphism, subtle gradients, and electric accent colors.
